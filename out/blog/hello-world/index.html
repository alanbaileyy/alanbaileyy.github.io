Hello, world

June 9, 2026

I've been meaning to build a personal site for a long time. This is it. The goal is simple: a quiet place to share what I build and write down what I learn.

Why bother

Writing forces clarity. When I try to explain something I half-understand, the gaps become obvious. A blog is just a public commitment to closing those gaps.

A few things I want to write about:

  • Small engineering decisions and the tradeoffs behind them
  • Tools I'm building and why
  • Notes from things I'm learning

How this works

Every post is a single Markdown file. To publish something new, I add a file and push to GitHub — the site rebuilds and deploys itself. No CMS, no database, nothing to maintain.

The best system is the one you'll actually use.
