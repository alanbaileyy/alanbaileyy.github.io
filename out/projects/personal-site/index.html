Personal Site

June 19, 2026

A minimal, file-based portfolio and blog that deploys itself to GitHub Pages.

This site. A minimal personal site built to be fast, calm, and effortless to update.

Goals

  • Zero maintenance. No CMS or database. Content is just Markdown files in the repo.
  • Fast. Statically exported HTML served from a CDN.
  • Pleasant to read. Generous whitespace, an editorial serif for headings, and a comfortable reading measure.

How it's built

The site is a Next.js app exported to static HTML. Posts and projects are MDX files with frontmatter; a small build-time helper reads them and generates the pages. A GitHub Action rebuilds and deploys on every push.

What I'd add next

  • A search box for posts
  • RSS feed generation
  • Optional dark mode

It's intentionally small — and that's the point.
